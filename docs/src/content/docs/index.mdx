## What is Infra Pilot?

Infra Pilot is a self-hosted infrastructure portal that brings a cloud-like deployment experience to your homelab. It integrates with Nomad, Cloudflare, Traefik, and Proxmox to give you a unified control plane.

<CardGrid>
  <Card title="App Platform" icon="rocket">
    Deploy Docker containers with a guided wizard. Search Docker Hub, configure resources, and deploy in minutes.
  </Card>
  <Card title="DNS Management" icon="setting">
    Create and manage Cloudflare DNS records. External apps auto-create A records pointing to your Nomad workers.
  </Card>
  <Card title="Job Orchestration" icon="seti:docker">
    Full Nomad job lifecycle management — deploy HCL specs, view allocations, stop, restart, and scale.
  </Card>
  <Card title="GitHub Integration" icon="github">
    Connect your GitHub account via the App Manifest flow. List repos and deploy from your own repositories.
  </Card>
